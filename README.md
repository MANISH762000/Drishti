# Drishti
An app for blind people 

# Features 

-voice assisted 

@ calculator

@ ocr to read any thing

@ current location

@ Battery percentage

@ currency detection using ocr

@ weather report

@ Date and Time 

@ Voice assisted command execution and also exit command 


*working app made and tested
